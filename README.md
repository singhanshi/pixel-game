# Pixel Game

A small 2D pixel-based game built using HTML5 Canvas and JavaScript.

The game explores movement, simple interaction, and spatial design through a compact indoor map. Two characters start in separate rooms and gradually find their way to each other through the environment.

---

## How to Play

- Click or tap anywhere on the screen to move
- Each input moves the character one tile
- Navigate through rooms and hallways
- When both characters reach the same tile, the game progresses

Designed to work on:
- Desktop browsers
- Mobile devices (touch supported)

---

## Features

- 2D pixel-art indoor environment
- Multi-room layout with hallways
- Click / tap based movement
- Two characters with customizable appearance
- Pixel furniture elements (table, sofa)
- Simple state-based progression
- Final screen triggered on character interaction

---

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- CSS
- Developed using CodePen
